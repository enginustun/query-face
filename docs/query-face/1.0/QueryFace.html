Class: QueryFace

This is the base class of query-face library that you can use to create ORM queries for backend with chainable methods.
Parameters:
Name Type Description
options QueryFaceOptions initial settings of query
Example
import QueryFace from 'query-face';
new QueryFace(options);
QueryFace(options); // shortened
// -----
const qf = QueryFace({ queryName: 'getUsers', dbName: 'myProject' });
qf.select('*').from('users');

// Suggested Usage
import qf from 'query-face';
qf({ queryName: 'getUsers', dbName: 'myProject' })
  .select('*')
  .from('users');

qf({ queryName: 'insertUser', dbName: 'myProject' })
  .insert({ name: 'engin', age: 28 })
  .into('users');

Methods

run(queriesopt, transactionopt, dependentopt, endpointopt, headersopt, methodopt, modeopt, cacheopt, credentialsopt, redirectopt, referreropt, beforeRunopt, afterRunopt) → {Promise.<Response>}

Sends query to server and fetches response
Parameters:
Name Type Attributes Default Description
queries Array <optional>
 [] for executing multiple queries
transaction boolean <optional>
 false activate transaction for multiple queries
dependent boolean <optional>
 false activate dependency for multiple queries
endpoint string <optional>
 Config.endpoint endpoint url to send request
headers object <optional>
 key-value header map
method string <optional>
 POST request method
mode string <optional>
 cors request mode
cache string <optional>
 no-cache request cache
credentials string <optional>
 include request credentials
redirect string <optional>
 follow request redirect
referrer string <optional>
 no-referrer request referrer
beforeRun function <optional>
 ()=>true function will be executed before run, if it returns false then process terminated
afterRun function <optional>
 ()=>{} function will be executed after run
Returns:
server's response as promise object, directly returns fetch's result
Type
Promise.<Response>
Example
const promiseResponse = qf().select('*').from('users').where('id', 1).run();
const users = promiseResponse.then(response => response.json());
// or
const response = await qf()
  .select('*')
  .from('users')
  .where('id', 1)
  .run();
const users = await response.json();
